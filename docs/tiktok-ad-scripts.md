# AgentVoz TikTok Ad Scripts

10 UGC-style hook videos for AgentVoz (agentvoz.com). $9.99/mo AI Spanish tutor.

**Format:** 3-5s hook (text overlay + actor reaction) + 10-15s product demo (Sofia speaking Spanish with karaoke subtitles).

**Demo recording needed:** Screen record of agentvoz.com showing Sofia conversation with karaoke subtitles active. Record 4 separate demos: Mexico dialect, Spain dialect, Argentina dialect, Nicaragua dialect. Portrait mode (phone filming laptop or phone screen). Each clip 15-20s raw, trim to 10-15s in post.

## Production Plan

### Recording the Demo Clips

Before generating any AI hooks, record 4 product demo clips on agentvoz.com:

1. **Mexico dialect demo** - Have a conversation with Sofia (Mexico). Show karaoke subtitles, show a beginner exchange, show her using colloquial phrases.
2. **Argentina dialect demo** - Switch to Argentine Sofia. Show the dialect difference clearly.
3. **Spain dialect demo** - Same pattern, Castilian Spanish.
4. **Nicaragua dialect demo** - Short clip for the 4-dialect montage.

Record each with phone filming the laptop screen (portrait, 1080x1920). Raw clips 15-20s each. Store in `~/Downloads/` or `~/A Team/content/agentvoz-demos/`.

### Generating Hook Videos with Veo 3

Use the existing pipeline at **`~/ateam-content/tiktok_ugc.py`**. This script:

1. Takes a hook text + model prompt per variant
2. Sends the model prompt to **Veo 3** (via Gemini API) to generate a 3-4s selfie-style reaction clip
3. Burns the hook text as a multi-line overlay using FFmpeg
4. Stitches the hook clip + demo recording into a final 1080x1920 TikTok-ready video
5. Outputs to `~/A Team/content/tiktok-ugc/`

**To adapt for AgentVoz:**

1. Copy and modify the `VARIANTS` list in `tiktok_ugc.py` (or create a parallel `tiktok_ugc_agentvoz.py` that imports the same generation/stitching functions).
2. Replace `DEMO_PATH` with the AgentVoz screen recording path.
3. Adjust `DEMO_TRIM_START` and `DEMO_TRIM_END` for each demo clip.
4. For scripts 3 and 8 (multi-dialect), you will need to manually stitch 4 short demo clips together first, then feed the combined clip as the demo input.
5. Run: `python tiktok_ugc.py --hook 1` (or whichever variant number) to generate one at a time, or no flag for all.

**Post-production:**
- Add trending audio in TikTok app (the pipeline outputs silent video intentionally)
- Each video should land between 13-20 seconds total
- Upload as TikTok ads via TikTok Ads Manager with "Spark Ads" format if using a creator account

### Testing Order

Start with scripts **1** (price), **2** (frustration), and **5** (authenticity) as the first batch. These are the strongest direct-response angles. If CTR > 2%, scale. If not, rotate to scripts 3, 4, 7.

Script 6 (emotional/grandmother) is a brand play. Run it separately to test for saves and shares rather than clicks.
